# Lisp for RT-11, БК-0010.01 & БК-0011М
LISP interpreter for RT-11 operating system and Soviet PDP-11-based БК-0010.01 & БК-0011М microcomputers.

## How to run

### RT-11
```
.RUN LISP
```

### БК-0010.01
```
MO
? M
ИМЯ? BKLISP.10
```

### БК-0011М
```
MO
@L
  Имя файла: BKLISP.11M
@G
```
